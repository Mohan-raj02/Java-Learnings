                String.valueOf(num)           // converted to string -- "12345"


2. To convert the string to integer:
                 eg: String str = "12345";     //string

                 Integer.parseInt(str)          // converted to integer -- 12345

 3. Difference Between Abstract Class & Interface
 Feature	           Abstract Class	                                           Interface
 Methods	  --Can have abstract + normal methods	            --Only abstract methods (till Java 7), from Java 8: default & static methods too
 Variables	  --Instance variables allowed	                    --Only public static final constants
 Inheritance  --One class can extend only 1 abstract class	    --One class can implement multiple interfaces
 Constructor  --Can have constructors	                        --No constructors
